## ⌨️ Installation
Preferences:

1. Go to `Extensions`
2. Search for `Chameleon Theme` and install it.
3. Go to `Color Themes` and choose `Chameleon` as the color theme or `CMD + Shift + P` and enter Command `> color theme` and choose `Chameleon`.
